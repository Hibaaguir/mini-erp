Mini-ERP
C'est un système ERP simple avec gestion des clients, produits et commandes.

1/Backend:
```bash
cd backend
npm install
npm start

2/Frontend:
   *Les fonctionnalités:
        Gestion des clients (Ajouter et consulter vos clients)
        
        Gestion des produits (Catalogue avec prix et stock)
        
        Gestion des commandes (Créer et suivre les commandes)
        
        Tableau de bord (Statistiques en temps réel)
        
        Modification des statuts (Suivi des commandes)
3/Utilisation:
   1-Démarrez le serveur : cd backend 
                          npm start
    2-Ouvrez l'interface : frontend/index.html
    3-Ajoutez des données : Utilisez le bouton "Charger données démo"
    4-Gérez votre activité : Naviguez entre les onglets
